# 💼 Employment Rating System

## 📖 Project Overview

This is my fourth Python mini project.

The application evaluates employee performance based on ratings and assigns an appropriate performance category using conditional statements.

---

## ✨ Features

- Accepts multiple employee records
- Evaluates performance based on ratings
- Assigns performance categories
- Displays employee details
- Generates a summary report

---

## 🛠️ Technologies Used

- Python 3
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements

---

## 📸 Sample Output

```
Employee Name : Rahul
Rating        : 4.5

Performance : Excellent
```
