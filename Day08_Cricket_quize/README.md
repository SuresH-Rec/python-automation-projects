# 🏏 Ultimate Cricket Stats Quiz

A console-based Cricket Quiz built using Python that tests your knowledge of cricket legends, records, milestones, and underrated domestic stars.

This project was created as part of my Python learning journey to practice functions, lists, dictionaries, loops, input validation, and modular programming while building something fun and interactive.

---

## 🚀 Features

- 🎯 Multiple Choice Questions (A/B/C/D)
- ✅ Input Validation
- 💡 Detailed explanation after every answer
- 📊 Final Score Calculation
- 🏆 Performance Grade based on your score
- 🔁 Interactive console-based gameplay

---

## 🎮 How to Play

1. Run the Python file.
2. Read each question carefully.
3. Enter your answer using:
   - A
   - B
   - C
   - D
4. Get instant feedback with explanations.
5. View your final score and performance grade at the end.
