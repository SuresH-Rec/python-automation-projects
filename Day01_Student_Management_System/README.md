# 🎓 Student Management System

## 📖 Project Overview

This is my first Python mini project.

The application stores student names and marks, calculates the result, and displays the appropriate grade.

---

## ✨ Features

- Accepts multiple students
- Calculates student result
- Displays:
  - Distinction
  - First Class
  - Passed
  - Fail
- Shows summary report
- Validates marks (0–100)

---

## 🛠 Python Concepts Used

- Functions
- Lists
- Dictionaries
- For Loop
- While Loop
- If / Elif / Else
- User Input
- Counters
- Input Validation
