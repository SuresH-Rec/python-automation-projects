# 🎯 Tech Recruiter Quiz

## 📖 Project Overview

This is my third Python mini project.

The application is a multiple-choice quiz designed for tech recruiters. It asks recruitment-related questions, validates user input, calculates the final score, and provides a grade with feedback.

---

## ✨ Features

- Multiple-choice quiz
- 10 recruitment-related questions
- Input validation (Accepts only A, B, C, or D)
- Displays explanation after each answer
- Calculates final score
- Assigns a grade based on performance
- Provides personalized feedback

---

## 🛠️ Technologies Used

- Python 3
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Input Validation
