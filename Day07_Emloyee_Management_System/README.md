# Employee Management System

A simple console-based Employee Management System built using Python.

This project allows users to manage employee records through a menu-driven interface. It is part of my Python learning journey, where I am practicing core programming concepts by building small real-world projects.

## Features

- Add Employee
- View All Employees
- Menu-driven interface
- Store employee records using lists and dictionaries

## Technologies Used

- Python 3

## Concepts Practiced

- Functions
- Lists
- Dictionaries
- Loops (`while`, `for`)
- Conditional Statements (`if`, `elif`, `else`)
- User Input
- f-Strings
- `enumerate()`

## Sample Menu

```text
===== Employee Management Menu =====
1. Add Employee
2. View Employees
3. Exit
```

## Learning Outcome

This project helped me understand how to:

- Organize code using functions
- Store multiple records using lists of dictionaries
- Display data in a structured format
- Build a simple menu-driven console application
